## Identifying the Xiaomi Yi hardware revision

The hardware revision can be identified based on the serial number:

| Serial          | Revision |
|-----------------|----------|
| Z221xxxxxxxxxxx | 22       |
| Z22Lxxxxxxxxxxx | 22L      |
| Z231xxxxxxxxxxx | 23       |
| Z23Lxxxxxxxxxxx | 23L      |
| Z23Axxxxxxxxxxx | 23A      |
| Z23Sxxxxxxxxxxx | 23S      |
